# PhishLab Images

This directory contains image assets for the PhishLab application.

## Placeholder Images

To enhance the visual appearance of PhishLab, consider adding:

- `banner.png` - Main banner image for README
- `logo.png` - Application logo
- `favicon.ico` - Browser tab icon
- `screenshot-*.png` - Application screenshots

## Image Guidelines

- Use PNG format for logos and icons
- Use JPEG for photos
- Optimize images for web (compress)
- Maximum width: 1200px for banners
- Use descriptive filenames

## Creating Images

You can create custom images using:
- Canva (https://canva.com)
- Figma (https://figma.com)
- Adobe Photoshop
- GIMP (free alternative)
